# BrightPath Coaching Classes – Responsive Education Landing Page

Link: https://its-aarav-codes.github.io/BrightPath-Coaching-Classes/

## Overview
BrightPath Coaching Classes is a **results-focused, single-page coaching institute website** designed for parents and students.  
The layout emphasizes **clarity, structure, and trust**, helping visitors quickly understand courses, timings, and outcomes—without confusion or unnecessary visuals.

This project demonstrates a **real-world front-end implementation** suitable for local coaching centers and educational institutes.

---

## Key Highlights
- Mobile-first, responsive layout
- Results-oriented hero section with academic statistics
- Clear course structure with batch timings
- Parent-focused content and trust signals
- Lightweight, fast-loading implementation
- Simple call-to-action via WhatsApp and email

---

## Tech Stack
- **HTML5** – semantic, accessible structure  
- **CSS3** – Flexbox and Grid for responsive layouts  
- **JavaScript (minimal)** – mobile menu toggle and smooth scrolling  

No backend, no frameworks, no heavy libraries.

---

## Sections Included
- Header & navigation (desktop + mobile)
- Hero section with performance highlights
- Courses offered with clear timings and structure
- Why parents choose us (trust & process)
- Student and parent testimonials
- Contact section with WhatsApp-first CTA

---

## Design Approach
The design focuses on:
- Clear information hierarchy for parents
- Practical layouts used by real coaching institutes
- Conservative visuals that build credibility
- Easy customization for different institutes

The goal is **usability and trust**, not flashy animations.

---

## How to Use
1. Clone or download the repository  
2. Open `index.html` in a browser  
3. Update text, course details, timings, and contact info  
4. Customize colors and fonts via CSS  
5. Deploy on any static hosting platform (GitHub Pages, Netlify, etc.)

---

## Customization Notes
- Course cards are built using CSS Grid for easy scaling
- Colors and typography are centralized in CSS
- Content can be reused for different education niches
- WhatsApp links can be updated with a single change
